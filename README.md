# Image Converter

一个简单的在线图片处理工具，支持图片压缩和格式转换。

## 功能特点

- 支持拖拽和点击上传图片
- 支持压缩到指定大小（200KB/500KB）
- 支持转换为 WebP 格式
- 实时预览和压缩率显示
- 支持点击预览区域更换图片

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- React Dropzone

## 本地开发

```bash
# 克隆项目
git clone https://github.com/goudan1030/image-converter.git

# 进入项目目录
cd image-converter

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 使用方法

1. 上传图片（支持拖拽或点击上传）
2. 选择输出格式（保留原格式或转换为WebP）
3. 选择目标大小（200KB或500KB）
4. 下载处理后的图片
